The code files are labeled according to the corresponding figures in the paper: 
compare_methods_nobs.py generates Figure 1.
compare_methods_gamma.py generates Figure 2.
compare_methods_noise.py generates Figure 3.
compare_methods_rank.py generates Figure 4.
comp_error.py generates Figure 5.
compare_ranks.py generates Figure 6.
consistency.py generates Figure 7.
computation_time.py generates Figure 8.

The file functions.py contains code for all the algorithms used, and a few utility functions.  

External dependencies may be installed using pip or conda.

All generated figures will be saved in the main folder.
